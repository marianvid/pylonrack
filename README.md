# PylonRack

A macOS menu bar application that acts as a universal launcher and monitor for independent services and applications.

Each slot in the rack holds one application. Applications communicate with the rack over WebSocket using the PylonRack protocol — language and runtime agnostic.

## Requirements

- macOS 13+
- Xcode 15+

## Building

Open `PylonRack/PylonRack.xcodeproj` in Xcode and build.
